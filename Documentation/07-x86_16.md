# x86_16 Architecture

## Overview

x86_16 is the 16-bit x86 real mode architecture used in legacy systems, bootloaders, and BIOS code.

## Register Set

### General Purpose Registers (16-bit)
- `ax`, `bx`, `cx`, `dx` - Accumulator, base, count, data
- `si`, `di` - Source and destination index
- `bp`, `sp` - Base pointer and stack pointer

### Segment Registers
- `cs`, `ds`, `es`, `ss` - Code, data, extra, stack segments

### Stack Simulation
- `bx` - Stack slot 0
- `cx` - Stack slot 1
- `dx` - Stack slot 2
- `di` - Stack slot 3

## Key Differences

1. **16-bit registers** - ax instead of eax
2. **Segmented memory** - 64KB segments
3. **Real mode addressing** - segment:offset
4. **No modern extensions**

## Calling Convention

```asm
push bp
mov bp, sp
; Function body
mov sp, bp
pop bp
ret
```

## Common Instructions

16-bit variants:
- `mov ax, bx`
- `add ax, 42`
- `push bp`
- `pop bp`
- `ret`

## Use Cases

- Bootloaders
- Real mode code
- Legacy system programming
- Educational purposes
